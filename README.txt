INTRUCCIONES DE INSTALACION
------------------------------------------------------------

Para instalar el sitio, siguir los siguiente pasos:

REQUISITOS
-----------------------------------------------------------
-Tener instalado Xammp o Wamp instalados 

--------------------------------------------------------------
-crear una base de datos llamada "servitec" en PHPMYADMIN MYSQL.

-cargar en PHPMYADMIN el documento llamado "php-jobsaid.sql"

-descargar o cargar el proyecto de JOBSAID desde github

-alojar la carpeta descargada en C:\xampp\htdocs.

-ejecutar desde el navegador http://localhost/JobsAid


USO
-------------------------------------------------------------
para saber como funciona cada parte de la apgina web te recomendamos leer el manual "manual de usuario"



CONTRIBUCIONES
-------------------------------------------------------------
